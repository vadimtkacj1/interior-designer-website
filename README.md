# Interior Design Landing Page

Современный landing page для дизайн-студии интерьеров, построенный на React, Vite и Tailwind CSS.

## 🚀 Технологии

- **React** - библиотека для создания пользовательских интерфейсов
- **Vite** - быстрый инструмент сборки
- **Tailwind CSS** - utility-first CSS фреймворк

## 📁 Структура проекта

```
src/
├── components/
│   ├── Hero/          # Главный баннер
│   ├── Features/      # Преимущества
│   ├── Services/      # Услуги
│   ├── Gallery/       # Портфолио проектов
│   ├── Contact/       # Форма обратной связи
│   └── Footer/        # Подвал сайта
├── assets/
│   └── images/        # Изображения
├── App.jsx            # Главный компонент
├── index.css          # Глобальные стили
└── main.jsx           # Точка входа
```

## 🎨 Секции Landing Page

1. **Hero** - Привлекательный заголовок с анимированным фоном
2. **Features** - Почему выбирают нас (4 ключевых преимущества)
3. **Services** - Услуги студии (6 карточек)
4. **Gallery** - Портфолио реализованных проектов
5. **Contact** - Форма обратной связи и контактная информация
6. **Footer** - Социальные сети и дополнительная информация

## 🎨 Кастомизация

### Цвета
Основные цвета настраиваются в tailwind.config.js

### Контент
- Тексты и заголовки находятся в соответствующих компонентах
- Изображения можно заменить в компоненте Gallery
- Контактную информацию обновите в Contact.jsx

### Темная тема
Проект поддерживает темную тему автоматически через prefers-color-scheme.
